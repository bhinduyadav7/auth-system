# 🔐 Auth System (Login , Signup & Dashboard )

A full-stack authentication system built using Java Servlets and MySQL with deployment on cloud.

# Live Demo

👉 https://auth-system-wpvt.onrender.com/signup.html

##  Tech Stack

* Java Servlets
* HTML / CSS
* Apache Tomcat
* MySQL (Railway Cloud Database)
* Docker
* Render (Deployment)

##  Features

* User Signup
* User Login
* Session-based Authentication
* Secure Dashboard Access
* Logout functionality
  

##  Application Flow

Signup → Login → Dashboard

##  Database

* Hosted on Railway (Cloud MySQL)
* Table: `users`
* Stores user credentials securely

##  Deployment

* Dockerized application
* Hosted on Render
* Connected with Railway database

##  Project Structure

* `src/com/app/servlet` → Java Servlets
* `WebContent` → HTML pages
* `Dockerfile` → Deployment config
* `.war file` → Packaged application
